# Values, Types and Operators

Try these expresions in a browser console and see if you can figure out why you get the results get:

* !!("")
* "" == false
* +true
* +false
* true + true
* 0 == "0"
* +"0"
* null == undefined
* null === undefined
* "foo" + + "bar"
* 9999999999999999
* 0.1 + 0.2 === 0.3
* 0.1 + 0.3
* 3 > 2 > 1
* 3 > 2 >= 1
* true > 1
* "2" + 1
* "2" - 1
* "2" -  - 1
* typeof NaN
